# animations_on_github_yearly_contributions_overview
Hacking the github yearly contributions overview grid to display text and animations with python script AND/OR a browser addon

# updates 
- started the repo on jan 1st 2021
- created griddy.py over a weekend of nov 2025

# how to install

```
- fork the repo or download the griddy.py script
- run griddy.py , no special dependencies
- give the text and a starting date (eg: 2017-01-01)
- go to your github homepage and go to the specified year
- tadaaaa 
- note : keep the text length < 7

```
